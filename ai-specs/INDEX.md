## Purpose

These specifications serve as:
- **Documentation** for future reference
- **Context** for AI agents working on related features
- **History** of AI-assisted development

## Reading Guide

Each spec file contains:
- Summary of the feature/improvement
- Detailed changes made to each module
- Configuration options and usage examples
- Testing information
- Future considerations

## Trivial Changes

Chores and trivial fixes are not documented here (files typically contain 10+ lines of substantive changes).
